# BUG-001 – Sample Defect Report

## Title

Cart badge does not update after adding a product to the cart.

## Environment

- Browser: Google Chrome
- Operating System: Windows 11
- Application: SauceDemo / Swag Labs
- Test Type: Functional Testing

## Severity

Medium

## Priority

High

## Preconditions

- User is logged in.
- User is on the Products page.
- Cart is empty before starting the test.

## Steps to Reproduce

1. Log in with a valid user.
2. Select one product.
3. Click Add to Cart.
4. Observe the cart badge in the top-right corner.

## Expected Result

The cart badge should display "1" after one product is added to the cart.

## Actual Result

The cart badge does not update after the product is added.

## Evidence

Screenshot should be added to the screenshots folder.

## Status

Open
