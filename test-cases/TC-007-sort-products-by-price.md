# TC-007 – Sort Products by Price

## Objective

Verify that the user can sort products by price from low to high.

## Preconditions

- User is logged in.
- User is on the Products page.

## Test Data

- Username: standard_user
- Password: secret_sauce
- Sorting option: Price (low to high)

## Test Steps

1. Log in with valid credentials.
2. Open the product sorting dropdown.
3. Select Price (low to high).
4. Review the product prices displayed on the page.

## Expected Result

Products should be displayed in ascending order, from the lowest price to the highest price.

## Actual Result

Products were displayed in ascending order by price.

## Status

Pass

## Notes

The sorting functionality worked as expected.
